DOM Manipulation Practice

Question #1: Use JS to write some HTML. Create a UL element and append it to the DOM as a child of the document body. Then create two LI elements, give them appropriate text content, and append them as children of the UL element. 


        const animalList = document.createElement("ul");
        document.body.appendChild(animalList);

        const item1 = document.createElement("li");
        item1.textContent = "fox";
        animalList.appendChild(item1);

        const item2 = document.createElement("li");
        item2.textContent = "hare";
        animalList.appendChild(item2);

Question #2: Create an array of animals. Then use a for...of loop to create and append each animal in the array as a listitem child of an UL element. Once this is working, add at least one additional item to the list. You should be able to do this by simply adding it to the array. 


        const animals = ["fox", "hare", "squirrel", "badger", "pony", "hedgehog"];
        
        const animalList = document.createElement("ul");
        document.body.appendChild(animalList);

        for (const animal of animals) {
            item = document.createElement("li");
            item.textContent = animal;
            animalList.appendChild(item);
        }

Question #3: Using the code created above, now write a line of code taht uses the removeChild() method and the lastChild property of the animalList element to remove only the last listitem element in the list. 


        animalList.removeChild(animalList.lastChild);

Question #4: Using the code created in #2 & #3, now write a line of code that uses removeChild() and the firstChild property of the animalList element to remove only the first listitem element in the list.
