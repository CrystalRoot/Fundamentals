Question #1: What will be the value of total once the below loop has finished running?


        let total = 0;
        let i = 1;
        while (i < 10) {
            total += i;
            i++;
        };

The value of total once the loop has completed will be 45.

In its first iteration, i = 1. Since 1 < 10, the condition is true, so the code inside the code block executes.

total =+ i adds the value of i (which in this case is 1) to total.

i++ increments i, so i is now 2.

Now i = 2. As 2 < 10, the code block executes again.

total =+ i This time total = 1, and i = 2. So we add 1 + 2, and total is now 3.

i++ increments i, so i is now 3.

And so on. The value of total will keep increasing by i in each iteration

After the final iteration, i will increment to 10, which no longer satisfy the condition, and the loop exits.

        };

        let total = 1;
        for (let i = 0; i < 10; i++) {

            };
        };
