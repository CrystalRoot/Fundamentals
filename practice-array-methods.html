Array Methods Practice

Question #1: Concat two arrays into a single array and assign the return value to a new array.


        const counties1 = ['Clare', 'Kerry', 'Galway']; 
        const counties2 = ['Donegal', 'Kilkenny', 'Meath'];
        const counties = counties1.concat(counties2);

Question #2: Create an array of animals and then sort it.


        const animals = ["fox", "hare", "squirrel", "badger", "pony", "hedgehog"];
        animals.sort();

Question #3: Now reverse the above array you created. 


        animals.reverse();

Question #4: Create an array with at least five items. Then use slice to remove the items in index 2 and 3 and place them in a new array.


        const instruments = ["fiddle", "whistle", "accordion", "mandolin", "bodhrán"];
        const newInstruments = instruments.slice(2, 4);

Question #5: Using the original array created in #4, add another item between index 2 and 3.


        const instruments = ["fiddle", "whistle", "accordion", "mandolin", "bodhrán"];
